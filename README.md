# Aventura dos Números

Projeto em Python feito para crianças usando **Tkinter**.

## Sobre o projeto
Aventura dos Números é um pequeno jogo educativo com interface gráfica. A criança pode praticar contas de **soma** e **subtração** de forma leve, colorida e divertida.

## Funcionalidades
- Interface gráfica feita com Tkinter
- Botão para gerar nova pergunta
- Dois modos de jogo: soma e subtração
- Campo para digitar a resposta
- Sistema de pontuação
- Contador de rodadas
- Mensagens de feedback quando a criança acerta ou erra
- Botão para reiniciar o jogo

## Biblioteca utilizada
- `tkinter`

## Como executar
1. Certifique-se de ter o Python 3 instalado.
2. Baixe os arquivos do projeto.
3. No terminal, execute:

```bash
python main.py
```

## Estrutura do projeto
- `main.py` -> código principal do jogo
- `requirements.txt` -> dependências do projeto
- `README.md` -> descrição do projeto e instruções

## Público-alvo
Crianças em fase inicial de aprendizagem de matemática e lógica.

## Objetivo pedagógico
Estimular raciocínio lógico, atenção e confiança ao resolver pequenos desafios matemáticos.
